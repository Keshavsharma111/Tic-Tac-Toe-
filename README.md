# Tic Tac Toe Game

A simple, interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project allows two players to play against each other on a 3x3 grid, featuring a clean UI and automatic win/draw detection.

## Features
*   **Responsive Design:** Works on different screen sizes [3].
*   **Two-Player Mode:** Alternates turns between 'X' and 'O' [3].
*   **Game Logic:** Detects win, loss, or draw conditions [2].
*   **Reset Functionality:** Easily restart the game [3].

## Tech Stack
*   **HTML5:** Structure of the game board [1].
*   **CSS3:** Styling and layout using Grid/Flexbox [2].
*   **JavaScript:** Game logic, DOM manipulation, and turn management [1].

## How to Play
1.  Open `index.html` in your web browser [11].
2.  Player 1 is 'X', Player 2 is 'O'.
3.  Click on the grid cells to place your mark [12].
4.  Get three in a row (horizontally, vertically, or diagonally) to win!
5.  Click the "Reset" button to start a new game [3].
